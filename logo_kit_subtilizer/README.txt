Pack de logos Subtilizer
========================

Ce dossier contient le logo officiel Subtilizer en plusieurs formats :

- subtilizer_logo.svg : version vectorielle, modifiable, HD
- subtilizer_logo_1024.png : PNG HD 1024x1024
- subtilizer_logo_512.png : PNG 512x512
- subtilizer_logo_256.png : PNG 256x256
- subtilizer_logo_128.png : PNG 128x128
- subtilizer_logo_64.png  : PNG 64x64
- subtilizer_logo_32.png  : PNG 32x32
- subtilizer_logo.ico     : Fichier icône multi-résolution (pour favicon, app Windows, etc.)

Utilisation :
- Web : utilisez le PNG ou le SVG dans vos assets, ou le .ico pour le favicon
- Desktop : utilisez le .ico pour l’icône de l’application
- Mobile : utilisez le PNG 512x512 ou 1024x1024

Le logo est libre d’utilisation pour tout projet Subtilizer.
